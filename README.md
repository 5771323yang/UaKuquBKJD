# 仓库管理系统简介 java202

## 项目概述

仓库管理系统是一款基于SpringBoot框架开发的前后端分离系统，旨在为用户提供高效、便捷的仓库管理解决方案。

## 技术栈

- 后端：SpringBoot、Mybatis-Plus、MySQL
- 前端：Vue、Element-ui

## 功能模块

### 系统角色

- 管理员（Admin）
  - 管理员与用户管理
  - 仓库信息管理
  - 物品分类管理
  - 物品信息管理
  - 查看操作日志

- 用户（User）
  - 物品信息查询
  - 查看操作日志

### 系统功能

- 仓库信息管理：新增、编辑、删除仓库
- 物品分类管理：新增、编辑、删除物品分类
- 物品信息管理：新增物品信息，指定仓库、分类、数量，物品入库、出库

## 运行环境

- 操作系统：Win10/11
- 开发工具：ideaIU-2022.3.3.win
- 开发语言：Java (jdk-8u371)、Vue(node.js 18.20)
- 项目管理工具：apache-maven-3.9.4
- 数据库：MySQL 8.0

## 部署步骤

1. 执行目录中的sql文件，创建数据库及表结构（Mysql用户名密码：root / root）
2. 解压项目压缩包
3. 使用IDEA导入项目
4. 等待Maven下载相关Jar包，下载完成之后点击启动项目
5. 在前端项目目录下执行`npm run dev`命令，启动前端工程

## 核心亮点

- 基于SpringBoot，系统具有良好的扩展性和稳定性
- 前后端分离，便于维护和升级
- 集成Mybatis-Plus，简化数据库操作，提高开发效率
- 支持多种运行环境，满足不同用户需求
